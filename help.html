Windows Phone 'Pin to start' web tile creator

help

You want to create your custom tile with a web link on your windows phone?

With this application you can config icon, background color and a label and pin to start it!

How it work:
  • insert the link you want to open
  • select the background color of your tile, 'theme' use default theme set on your phone
  • select an icon or your own image using 'custom image' (best if 320x320 .pgn with transparent background) or search online using 'search image'
  • add an optional label at the bottom of the tile
  • create it!
  • if you like it, from the browser menu '...' select 'Pin to start'
  • done
